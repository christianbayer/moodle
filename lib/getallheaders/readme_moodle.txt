getallheaders
===

This library contains the getallheaders package.


Package Version
---------------

- ralouphie/getallheaders: 3.0.3


Modifications
-------------

/lib.php

    Include the getallhedaers package file. Every use of the package should
    start by requiring this file:

        require_once($CFG->libdir . '/getallheaders/lib.php');

    And, from there, use the package normally. Everything will be autoloaded.
